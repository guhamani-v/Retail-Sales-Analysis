# Retail Sales Performance Analysis

## Project Overview

This project analyzes retail sales data to identify business trends, customer purchasing behavior, and product performance.

## Business Problem

RetailHub wants to understand:

- Which products perform the best
- Which cities generate the highest revenue
- Monthly sales trends
- Customer purchasing patterns
- Actionable recommendations for improving sales

## Tools Used

- SQL
- Microsoft Excel
- Power BI
